#  SportZone - DecodeLabs Frontend Internship Project

##  Project Overview

**SportZone** is a comprehensive frontend development project showcasing progressive web development skills. This project demonstrates a complete learning journey from fundamental HTML/CSS to advanced JavaScript, featuring a fully interactive sports website with modern design patterns and best practices.

Created as part of the **DecodeLabs Frontend Internship 2026**, this project includes three progressive tasks that build upon each other, each introducing new concepts and technologies.

##  Tasks Overview

###  Task 1: Basic Sports Website
**Level:** Beginner | **Duration:** 1-2 hours  
**Focus:** HTML5 & CSS3 Fundamentals

A clean, simple sports website showcasing cricket and football players. This task focuses on understanding HTML structure and basic CSS styling.

**Features:**
- Semantic HTML5 markup
- Navigation and hero section
- Player cards display
- Statistics section
- Contact section with footer
- Responsive basics

**Tech Stack:** HTML5, CSS3

 **Location:** [task-1-upendra/](./task-1-upendra/)

---

###  Task 2: Enhanced Sports Website
**Level:** Intermediate | **Duration:** 2-3 hours  
**Focus:** Responsive Design & Advanced CSS

An enhanced version with better styling, player statistics, section descriptions, and improved responsive design. This task introduces advanced CSS techniques and mobile-first approach.

**Features:**
- Responsive design (mobile, tablet, desktop)
- Enhanced player cards with stats
- Section descriptions
- Professional styling
- Flexbox and Grid layouts
- Better user experience

**Tech Stack:** HTML5, CSS3 (Advanced)

 **Location:** [task-2-upendra/](./task-2-upendra/)

---

###  Task 3: Advanced Interactive Website
**Level:** Advanced | **Duration:** 4-6 hours  
**Focus:** JavaScript & Interactive Features

A fully-featured, production-ready website with interactive elements, dark mode, quiz functionality, search capabilities, and smooth animations. This task demonstrates modern web development practices.

**Features:**
- Dark mode / Light mode toggle
- Interactive quiz game
- Search and filter functionality
- Smooth animations and transitions
- Typing animation effect
- Hamburger menu for mobile
- News section
- Local storage implementation
- Responsive on all devices

**Tech Stack:** HTML5, CSS3, JavaScript ES6+

**Location:** [task-3-upendra/](./task-3-upendra/)
