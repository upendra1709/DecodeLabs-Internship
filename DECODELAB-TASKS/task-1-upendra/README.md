# SportZone - Task 1: Basic Sports Website

##  Overview

SportZone Task 1 is a **clean and simple sports website** showcasing cricket and football players. It's designed to be beginner-friendly with straightforward HTML structure and CSS styling.

##  Features

- **Hero Section** - Welcome banner with call-to-action button
- **Cricket Section** - Display of top Indian cricket players
  - Virat Kohli
  - Rohit Sharma
  - Jasprit Bumrah
- **Football Section** - Showcase of legendary football players
  - Lionel Messi
  - Cristiano Ronaldo
  - Kylian Mbappé
- **Sports Statistics** - Fun facts about cricket and football
- **Simple Navigation** - Easy-to-use menu with smooth scrolling
- **Responsive Design** - Works well on different screen sizes

##  How to Use

1. **Open the Website**
   - Simply double-click `index.html` or open it in your browser
   - No server required!

2. **Navigate Through Sections**
   - Click on navigation links to jump to different sections
   - Or scroll down to explore content

3. **Explore Player Information**
   - View player cards with images
   - Read their roles and country information
