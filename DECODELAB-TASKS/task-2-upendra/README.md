# SportZone - Task 2: Enhanced Sports Website

##  Overview

SportZone Task 2 is an **enhanced sports website** with more detailed player information, advanced styling, and a polished user interface. This version includes player statistics, descriptive sections, and improved visual hierarchy.

##  Features

- **Responsive Header** - Modern navigation with hamburger menu support
- **Enhanced Hero Section** - More engaging welcome with better spacing
- **Detailed Player Cards** - Rich information display with:
  - Player images
  - Role tags (Batsman, Opener, etc.)
  - Player names
  - Career statistics
  - Country information
- **Section Descriptions** - Context about each sports category
- **Advanced Statistics** - "Sports by Numbers " section with:
  - Test Matches Played
  - FIFA World Cups
  - And more
- **Interactive Contact Section** - Reach out to the team
- **Social Media Links** - Connect on @SportZoneIndia
- **Professional Footer** - Polished credit section

##  How to Use

### Opening the Website
1. Navigate to the task2 folder
2. Double-click `index.html` or right-click → "Open with Browser"
3. No server installation needed!

### Navigating Content
- Use the navigation menu to jump to different sections
- Click on cards to view player details (if JavaScript is enabled)
- Scroll smoothly through all sections
- Click "Reach out" to contact the team

### Exploring Player Information
- Hover over cards to see visual effects
- Read player statistics and achievements
- View career highlights
